This code is based on DeOldify, an existing tool that can be found at https://github.com/jantic/DeOldify?tab=readme-ov-file#about-deoldify

In order to train the model, use the command python ColorizeTrainingNoWandb.py
Add color and b&w pairs to the 'training_images' folder.

Run python ImageColorizer.py to run the image colorizer.
